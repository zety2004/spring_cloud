#### 软件架构
软件架构说明
spring cloud 架构
由zuul路由，springConfig 配置中心，eureka注册中心组成，集成docker+jenkins自动化部署
例子中有两个微服务项目可以相互调用

#### 安装教程

系统环境jdk1.8,docker远程服务，jenkins远程服务,本地git服务

2. springconfig
3. 其他不限定
